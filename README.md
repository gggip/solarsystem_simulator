# 🌍 Interactive Solar System Explorer

An interactive educational web experience built with Three.js and AI-assisted workflows.

This project started as a simple experiment to help children explore the solar system in a more immersive and playful way — then gradually evolved through community feedback and iteration.

## ✨ Features

* 🪐 Interactive planet exploration
* ☄️ Asteroid belt
* 🔍 Space investigation missions
* 📚 Planet encyclopedia
* 📱 iPad-friendly interaction
* 🌌 3D space environment using Three.js

## 🎯 Project Goal

Instead of presenting space as static information, this project explores how interactive experiences can encourage curiosity, exploration, and self-directed learning for children.

* JavaScript
* Three.js
* AI-assisted development workflow
